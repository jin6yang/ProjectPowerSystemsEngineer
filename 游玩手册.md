# ProjectPSE - 游玩手册

> **前言**
> 
> 这是一个体素风格的模拟经营，解密游戏。



## 游戏背景

大型星际企业黑曜石（OBSIDIAN.INC）旗下的黑曜石工业（OBSIDIAN Industrial）正在招聘新的员工，玩家是前来应聘的普通人，应聘岗位为“能源及电网系统工程师”，玩家需要操作入职考核系统，通过考核后，就可以作为正式员工加入公司。



## 基本规则

1. 玩家需要通过控制 `功率（MW）` 和 `稳定度值（STB）` 来完成考核目标

2. 游戏关卡在一个沙盒上进行，沙盒会被网格系统分割成若干格子，格子中可以由玩家摆放建筑

3. 发电类型的建筑之间无法直接相连，必须通过其它建筑中转

4. 分布式储能，多个相连的储能建筑视为一个大的资源池，而一条电网中的多个储能集群视为不同的资源池

5. 如果建筑或者设备超过承载上限，则自动断电保护
   
   

## 操作手册

> 当前游戏**不支持**自定义改建

### 键盘和鼠标

| 操作     | 键盘    | 鼠标                              |
| -------- | ------- | --------------------------------- |
| 移动视角 | W/S/A/D | 左键按住拖动/鼠标光标撞击屏幕边框 |
| 缩放视角 |         | 滚轮                              |
| 旋转视角 | Q/E     | 右键按住拖动                      |
| 删除建筑 | DEL     |                                   |

### 手柄

> **推荐**使用`Steam Controller`，或者进行自定义**映射**

| 操作     | 手柄           |
| -------- | -------------- |
| 移动视角 | 右摇杆         |
| 缩放视角 | RT/LT          |
| 旋转视角 | 按住右摇杆推动 |
| 删除建筑 |                |

3. 按住 `~` 键 `3s` 当前关卡直接任务失败
